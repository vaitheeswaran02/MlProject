**🚗 Car Price Prediction System**

Overview
This project is a Machine Learning-based Car Price Prediction System that estimates the price of a car based on user inputs such as model, year, fuel type, and other features.
It provides a user-friendly UI where users can enter car details and get an approximate price instantly.


**📊 Input Features**
The model takes the following inputs:
Car Name / Model
Year of Purchase
Fuel Type
Seller Type
Transmission
Kilometers Driven
Owner Type

**🛠️ Technologies Used**
Python 🐍
Pandas & NumPy
Scikit-learn
Streamlit
